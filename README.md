## API Reference:

### Singly Linked List:
<a name="#singly-linked-list"></a>
#### Methods:
1. append(data) -> None
   - append data to the end of the list
2. append_at_location(data, location) -> None
   - append data to the location of the list
3. search(data) -> bool 
   - search for data in the list
4. remove(data) -> None 
   - remove data from the list
5. clear() -> None 
   - clear the list

### Doubly Linked List:
<a name="#doubly-linked-list"></a>
#### Methods:
1. append(data) -> None
   - append data to the end of the list
2. append_at_location(data, location) -> None
    - append data to the location of the list
3. insert(data, location) -> None
    - insert data to the front of the list
4. search(data) -> bool
    - search for data in the list
5. remove(data) -> None
    - remove data from the list
6. clear() -> None
    - clear the list
